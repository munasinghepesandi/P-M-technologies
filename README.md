<h1>My Official Website - P&M Technologies</h1>
<h2>My introductional website using React.js + Tailwindcss + JavaScript + PHP + MySQL</h2>
<p>This site introduce me and my hands on experiences. This site includes Home page, About me, Education, Soft skills, Featured Products and contact me sections. As Frontend tecnologies I used mainly Reactjs, JavaScript and as a framework I used TailwindCSS. As the Backend technologies I used PHP to connect with database with using MySQL queries. Inadvance, I used Ajax technology to perform actions in the background without reloading the entire web page, which creates a faster and more user-friendly experience. </p>
